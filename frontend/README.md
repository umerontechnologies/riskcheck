# RiskCheck Frontend (Vite + React)

## Setup

```bash
cd frontend
npm install
```

## Run (dev)

```bash
npm run dev
```

The dev server runs on http://localhost:5173

By default, Vite proxies `/api` to `http://127.0.0.1:8000` (see `vite.config.js`).

## Build (production)

```bash
npm run build
```

The static build will be created in `frontend/dist`.
